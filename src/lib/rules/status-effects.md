| Status Effect | Description                                                        |
|---------------|--------------------------------------------------------------------|
| **Dazed**     | Temporarily reduces your **Insight** die size by one               |
| **Enraged**   | Temporarily reduces **Dexterity** and **Insight** die sizes by one |
| **Poisioned** | Temporarily reduces **Might** and **Willpower** die sizes by one   |
| **Shaken**    | Temporarily reduces **Willpower** die size by one                  |
| **Slow**      | Temporarily reduces **Dexterity** die size by one                  |
| **Weak**      | Temporarily reduces **Might** die size by one                      |
